# 今天吃什么微信小程序

## 打开方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择：`/Users/macminim4/Documents/New project/food`
4. AppID 使用：`wx7b730fa79ce57b66`
5. 导入后编译预览。

## 当前功能

- 首页填写家庭情况、预算、口味、买菜方式。
- 支持“几点前做好”和“做饭速度”。
- 生成三套晚餐方案。
- 展示买菜清单、按点上桌计划、做饭顺序、每道菜做法。
- 本地记住上次设置和上次选中的一桌菜。

## 后续接服务器

当前生成逻辑是本地 mock 数据，位于 `utils/mealPlans.js`。
接云服务器后，把结果页里的本地生成替换为 `wx.request` 即可。
